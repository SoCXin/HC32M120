[HC32M120](https://github.com/SoCXin/HC32M120)

* 4KB SRAM + 32KB Flash
* 2通道DMA，每个通道触发源可从外设事件中独立选择
* 最多12通道12bit 1MSPS ADC，支持单次/扫描变换，支持各个Timer事件触发开始变换
* 2个独立运算放大器（OPA），支持最大驱动电流达30mA
* 2个电压比较器（CMP），支持窗口比较功能
